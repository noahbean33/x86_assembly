# Why Learn Assembly

Whether you're an embedded developer bringing up a microcontroller or a web developer writing a website using the latest Javascript Framework, understanding how computers work at a fundamental level will make you a better programmer.

Computers are simple, and assembly language is even simpler.

This course will teach you from start to finish the basics of computer architecture, how the ARM architecture is organized, and how to write programs in ARM assembly.

## Importance of ARM Assembly in Modern Computing

### Why ARM?

Out of all the computer architectures out there, I highly recommend learning ARM architecture first. Why is that? It's boils down to three reasons for me.

### Computation

While Intel is the most pervasive CPU architecture on the market, it's also the most power hungry. This makes sense, given that the architecture is a Complex Instruction Set Computer (CISC). These CPU cores are seen everywhere, from desktop computers to datacenters. ARM is a more computationally efficient architecture, able to perform the same kinds of instructions with much less energy.

### Learning

Learning is hard, and learning assembly can be even harder. You could be learning a more common instruction set like Intel, but the architecture is so complex that I think it would scare people off. On the other hand, you could be learning an older microcontroller architecture like PIC or Z80. However, those are harder to find development boards for, and aren't nearly as popular. ARM strikes a balance between accessibility and ease of use.

### Market Share

Because other Reduced Instruction Set Computer (RISC) architectures like ARM are becoming more powerful while not emitting as much heat, ARM CPUs are becoming more popular in regular computing, like phones and laptops.
